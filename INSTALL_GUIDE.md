# Installation Guide - Mobile Client Mod v26.23

## 🔧 Installation Methods

### ✅ EASIEST: Direct .mcpack Installation

**Windows 10/11:**
1. Download `MobileClientMod.mcpack`
2. Double-click the file
3. Minecraft will launch automatically
4. Click "Create" or "Edit World"
5. Select the pack from the list
6. Done! ✓

**Mobile (iOS/Android):**
1. Download `MobileClientMod.mcpack` 
2. Tap to open with Minecraft
3. Select "Create" when prompted
4. The pack will be imported
5. Done! ✓

---

### ⚡ ALTERNATIVE: .mcaddon Installation

**Windows 10/11:**
```
C:\Users\YourUsername\AppData\Local\Packages\Microsoft.MinecraftUWP_8wekyb3d8bbwe\LocalState\games\com.mojang\
```

1. Download `MobileClientMod.mcaddon`
2. Extract/Copy to: `behavior_packs/` folder (see path above)
3. Open Minecraft
4. Go to Settings → Global Resources
5. Find "Mobile Client Mod" and activate it
6. Create a new world and enable it
7. Done! ✓

**Android:**
```
Android/data/com.mojang.minecraftpe/files/games/com.mojang/
```

1. Download to your device
2. Use a file manager to navigate to above path
3. Place in `behavior_packs/` folder
4. Open Minecraft and enable in world settings

**iOS:**
- Download and tap file
- Select "Open with Minecraft"
- Import automatically

## 🎮 First Time Setup

1. **Launch Minecraft**
2. **Create a new world** (or edit existing)
3. **Enable the addon** in world settings
4. **Experimental features** should be ON (auto-enabled)
5. **Play!** - Press Sneak to toggle UI

## ⚡ Quick Start

1. Install mod (see above)
2. Create world with mod enabled
3. Press **Sneak** to open main menu
4. Select **FPS/CPS Display** to see stats
5. Go to **Settings** to customize

## ❌ Common Issues

### "Pack not found"
- Ensure file is in correct folder (see paths above)
- Try renaming to simpler name (no spaces)
- Restart Minecraft completely

### "UI not appearing"
- Make sure you're **actually sneaking**
- Check experimental features are enabled
- Try reinstalling the addon

### "Low FPS"
- Disable animations in settings
- Reduce render distance
- Use Optifine mod alongside

### "Cape not showing"
- Install 18.5 texture pack
- Enable PvP textures in mod settings
- Texture pack must support capes
