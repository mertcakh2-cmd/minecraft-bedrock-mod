## 🏃 How to Install

### Method 1: Direct Installation
1. Download the `.mcpack` file
2. Double-click the file
3. Minecraft will automatically import it
4. Select "Create" and choose the pack

### Method 2: Manual Installation
1. Download the `.mcaddon` file
2. Place it in: `%LOCALAPPDATA%\Packages\Microsoft.MinecraftUWP_8wekyb3d8bbwe\LocalState\games\com.mojang\behavior_packs/`
3. Launch Minecraft and enable in world settings

### Method 3: Mobile (Pocket Edition)
1. Download on your mobile device
2. Tap the file to open with Minecraft
3. Select "Create" or "Edit World"
4. Enable the addon in pack settings

## 🐛 Troubleshooting

**UI not appearing?**
- Make sure you're sneaking (press Sneak key)
- Check that experimental features are enabled

**Low FPS?**
- Reduce animation speed in settings
- Disable unnecessary visual effects
- Lower render distance in Minecraft settings
- Install Optifine-like optimization mods

**Cape not showing?**
- Download compatible 18.5 texture pack
- Ensure PvP textures are enabled
- Re-apply cape from settings
